# KA-Agent 查詢及檢索測試劇本

**創建日期**: 2026-01-28
**創建人**: Daniel Chung
**最後修改日期**: 2026-01-28

## 概述

本文檔提供 KA-Agent 查詢及檢索的測試劇本，包含 20 種不同類型的問法，用於驗證 KA-Agent 的檢索能力、錯誤處理和用戶反饋機制。

重要提醒，若要測試或修改，請在venv環境！

## 測試分類

測試劇本分為以下類別：

- **基礎查詢類**: 簡單的文件數量和列表查詢
- **內容檢索類**: 根據內容關鍵詞檢索知識
- **領域專項類**: 特定領域或主題的查詢
- **多條件查詢類**: 組合多個條件的複雜查詢
- **邊界情況類**: 空查詢、錯誤參數等
- **錯誤場景類**: 權限不足、資源不存在等

## 測試環境準備

### 前置條件

- ✅ KA-Agent 已註冊並在線
- ✅ 知識庫中已有測試數據（至少 10+ 文件）
- ✅ 向量化服務正常運行
- ✅ 用戶權限已正確配置

### 測試數據建議

建議準備以下類型的測試文件：

1. 技術文檔（API 規格、開發指南）
2. 產品文檔（使用手冊、規格說明）
3. 項目文檔（需求文檔、設計文檔）
4. 操作文檔（流程圖、SOP）
5. 知識圖譜數據（實體、關係）

**測試 ID**: KA-TEST-001

**優先級**: P0（高）

**用戶查詢**:

## 自動化測試建議

### 單元測試（Unit Test）

```python
import pytest
from agents.builtin.ka_agent.agent import KnowledgeArchitectAgent
from agents.services.protocol.base import AgentServiceRequest

@pytest.mark.asyncio
async def test_empty_query_error():
    """測試空查詢的錯誤處理"""
    agent = KnowledgeArchitectAgent()
    request = AgentServiceRequest(
        task_id="test_001",
        task_type="query",
        task_data={"instruction": ""},  # 空查詢
        metadata={"user_id": "test_user"},
    )
  
    response = await agent.execute(request)
  
    assert response.status == "failed"
    assert "缺少必要參數" in response.error
    assert "instruction" in response.error

@pytest.mark.asyncio
async def test_no_results_feedback():
    """測試未找到結果的反饋"""
    agent = KnowledgeArchitectAgent()
    request = AgentServiceRequest(
        task_id="test_002",
        task_type="query",
        task_data={"instruction": "查詢不存在的內容"},
        metadata={"user_id": "test_user"},
    )
  
    response = await agent.execute(request)
  
    assert response.status == "completed"
    assert "沒有找到" in response.result.get("message", "")
    assert "回問" in response.result.get("message", "")
```

import pytest
from fastapi.testclient import TestClient
