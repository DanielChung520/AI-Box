## 📋 概述

生成式AI链式处理系统是 AI-Box 的核心处理流程，包括 System Prompt 角色定义、文件上传与处理、向量化、知识提取等端到端的处理链路。

---

## 🔧 核心功能

### 1. System Prompt 角色定义

**功能**：通过Prompt定义AI的角色与技能

**应用场景**：

- 助理（Assistant）角色定义
- 代理（Agent）角色定义
- 任务特定的角色定义

### 2. 文件处理流程

**处理阶段**：

1. **文件上传**：多格式文件上传（PDF、Word、Excel、Markdown等）
2. **文件解析**：根据文件类型选择对应解析器
3. **语义切片**：通过语义理解进行文档切片
4. **向量化**：生成向量并存储到ChromaDB
5. **知识提取**：NER/RE/RT三元组提取，存储到知识图谱
