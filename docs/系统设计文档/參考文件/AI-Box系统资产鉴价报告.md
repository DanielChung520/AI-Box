## 一、系统概述

### 1.1 系统定位

**AI-Box** 是一个**集成式企业AI平台**，整合了以下核心能力：

- **生成式AI**：多模型管理与路由（MoE架构）
- **RAG系统**：基于向量数据库的语义检索
- **知识图谱**：基于图数据库的知识关联与推理
- **Agent平台**：多Agent协调与任务编排
- **AAM系统**：长短记忆架构的上下文管理
- **IEE前端**：Intelligent Editor Environment 专业编辑界面
- **Ontology系统**：3-tier架构的知识本体管理

### 1.2 技术架构复杂度

- **分层架构**：前端层、编排层、Agent层、GenAI层、记忆层、知识层、数据层、基础设施层（8层架构）
- **技术栈广度**：Python (FastAPI) + TypeScript/React + ChromaDB + ArangoDB + Redis + Kubernetes
- **集成复杂度**：多数据库集成、多LLM Provider集成、异步任务处理、容器化部署
