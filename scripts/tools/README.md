# 工具腳本目錄

**創建日期**: 2026-01-05
**創建人**: Daniel Chung

## 📋 目錄說明

本目錄包含各種開發和維護工具腳本。

---

## 📁 文件列表

### 代碼質量檢查

- **`check_code_quality.py`**: 自動化代碼質量檢查腳本
- **`check_docs_references.py`**: 文檔引用檢查腳本

### 代碼註冊表管理

- **`generate_code_registry.py`**: 生成代碼註冊表
- **`convert_code_registry_to_excel.py`**: 將代碼註冊表轉換為 Excel
- **`update_related_files_in_excel.py`**: 更新 Excel 中的相關文件
- **`update_tools_code_registry_full.py`**: 更新工具代碼註冊表（完整版）

### API 和工具同步

- **`sync_tools_via_api.py`**: 通過 API 同步工具

### 文檔處理

- **`vectorize_system_docs.py`**: 向量化系統文檔

### Ontology 管理

- **`import_ontology.py`**: 將 Ontology JSON 文件導入到 ArangoDB

### 文件檢查

- **`check_files.js`**: 文件檢查腳本（JavaScript）
