# 文件上傳說明

由於 API 服務可能未運行，請選擇以下方式之一上傳文件：

## 方式一：通過前端界面上傳（推薦）

1. 啟動前端應用和 API 服務
2. 登入系統
3. 上傳文件：`docs/东方伊厨-预制菜发展策略报告20250902.pdf`
4. 獲取文件 ID（file_id）

## 方式二：使用上傳腳本

確保 API 服務運行後，執行：

```bash
python test_upload_prepared_food.py
```

## 方式三：如果文件已上傳

如果文件已經通過其他方式上傳，可以直接使用 file_id 繼續後續測試步驟。

---

**注意**：上傳完成後，請記錄 file_id，以便執行後續的向量化驗證和圖譜提取測試。
