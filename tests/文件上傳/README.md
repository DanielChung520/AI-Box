## 📝 文件分類說明

### 測試腳本 (`test_*.py`, `update_*.py`, `upload_*.py`)

- **用途**: 執行各種測試的 Python 腳本
- **說明**: 包括文件上傳、向量化驗證、圖譜提取、模型對比等測試

### 模型對比報告 (`模型對比報告/`)

- **內容**: 不同 LLM 模型的性能對比測試報告
- **格式**: Markdown + JSON
- **包含**: mistral-nemo:12b, qwen3-next:latest, gpt-oss:120b-cloud 等模型的測試結果

### 圖譜重新生成 (`圖譜重新生成/`)

- **內容**: 圖譜重新生成測試的結果報告
- **格式**: JSON

### 時間記錄 (`時間記錄/`)

- **內容**: 文件處理各階段時間記錄
- **格式**: Markdown + JSON
- **包含**: 分塊、向量化、存儲、圖譜提取等各階段時間統計

### 問題分析 (`問題分析/`)

- **內容**: 問題分析、排查和修復報告
- **格式**: Markdown
- **包含**: 圖譜提取為 0、NER/RE/RT 問題、模型配置問題等分析報告

### 測試日誌和結果 (`*.log`, `*.json`)

- **用途**: 測試執行過程的日誌和結果數據
- **格式**: 文本日誌和 JSON 結果

### 測試計劃和總結 (`*.md`)

- **內容**: 測試計劃、測試總結、測試說明文檔
- **格式**: Markdown
